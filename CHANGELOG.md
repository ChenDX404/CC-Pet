# Change Log

## 0.1.0

- 提供 VS Code 侧边栏动作绑定与人物配置。
- 提供 Tauri 透明桌面浮窗。
- 支持 Claude Code 工作状态、问题气泡与最终回复气泡。
- 支持拖动、点击、悬浮、出现、空闲、工作和完成事件。
- 支持人物目录扫描、人物选择、显示大小及逐动作速度设置。
- 增加独立桌面设置窗口、系统托盘和可选开机启动。
- 将用户配置迁移到 `%APPDATA%\CC Pet\config.json`。
- 每个 Codex pet 使用 `%APPDATA%\CC Pet\profiles\<文件夹名>.json` 独立保存动作与事件绑定，并自动迁移旧配置。
